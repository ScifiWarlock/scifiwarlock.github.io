MathSearch!

MathSearch is a new feature I have made that takes my notes and lists lemmas, theorems, definitions, etc. It is still a heavy work in progress, but it can be used to quickly search up topics for example "Bolzano-Weierstrass" or "topology"! Some LaTeX symbols may not appear properly due to limitations of MathJAX3, but I will be updating the search to work with MathJAX4 which supports more symbols.
